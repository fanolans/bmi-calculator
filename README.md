## Run Locally
Clone the project

```bash
  git clone https://github.com/fanolans/bmi-calculator.git
```

Go to the project directory

```bash
  cd project-folder
```

Install dependencies

```bash
  flutter pub get
```

Start the app

```bash
  flutter run
```
